# AbsENS Dashboard

Application web de gestion des absences des étudiants développée avec Angular.  
Elle permet de suivre, analyser et gérer efficacement les absences au sein d’un établissement éducatif.

## Fonctionnalités
- Gestion des étudiants (ajout, modification, suppression)
- Saisie et suivi des absences
- Tableau de bord avec statistiques (taux d’absentéisme, alertes…)
- Filtres avancés et historique des absences
- Génération de rapports

## Technologies
- **Angular** : framework front-end pour créer une interface dynamique et moderne
- **TypeScript** : langage utilisé pour structurer et sécuriser le code
- **JSON Server** : API REST simulée pour gérer les données (backend léger)

## Objectif
Ce projet a pour but de mettre en pratique le développement d’une application complète avec gestion des données, interface utilisateur et logique métier.
